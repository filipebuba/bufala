## 📋 Como Usar

### Flutter
```yaml
flutter:
  assets:
    - assets/icons/
```

### Android
Copie os arquivos para:
- `android/app/src/main/res/mipmap-*/`

### iOS
Adicione ao `ios/Runner/Assets.xcassets/AppIcon.appiconset/`

### Web
Copie para `web/` e referencie no `index.html`
